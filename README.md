# Берёзовый Сок - Ultra Modern Website

Ультрасовременный одностраничный сайт о берёзовом соке с горизонтальным скроллом, 3D анимациями и интерактивными эффектами.

![Preview](preview.jpg)

## Особенности

### 🎨 Дизайн
- **Горизонтальный скролл** - уникальная навигация между секциями
- **3D анимации** - плавающая бутылка с жидкостью, парallax-эффекты
- **Кастомный курсор** - интерактивный курсор с эффектом наведения
- **Стеклянный морфизм** - современные эффекты размытия и прозрачности
- **Анимированные градиенты** - живые фоны и переходы

### ⚡ Функциональность
- **7 секций**: Hero, О продукте, Польза, Галерея, Видео, Игра, Контакты
- **Адаптивный дизайн** - работает на всех устройствах (desktop, tablet, mobile)
- **Кроссбраузерность** - поддержка Chrome, Firefox, Safari, Edge
- **Оптимизация производительности** - lazy loading, debounced events
- **Доступность** - поддержка prefers-reduced-motion

### 🎯 Интерактивные элементы
- **Слайдер галереи** - Swiper с эффектом fade и thumbs
- **Lightbox** - просмотр фотографий в полном размере
- **Видео модал** - всплывающее окно для видео
- **Анимированные счётчики** - статистика с анимацией чисел
- **3D наклон карточек** - эффект tilt при наведении
- **Прогресс-индикатор** - навигация по секциям

### 📱 Технологии
- **HTML5** - семантическая разметка
- **CSS3** - Grid, Flexbox, Custom Properties, Animations
- **JavaScript** - ES6+, GSAP, ScrollTrigger
- **Three.js** - готов к 3D расширениям
- **Swiper** - слайдер галереи
- **Font Awesome** - иконки
- **Google Fonts** - Playfair Display + Inter

## Установка и запуск

1. Клонируйте или скачайте репозиторий
2. Откройте `index.html` в браузере
3. Для разработки используйте live server

```bash
# Установите live-server глобально
npm install -g live-server

# Запустите
cd birch-sap
live-server
```
